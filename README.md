# techdoc-md
Tool for producing technical documentation using Markdown and Pandoc

## Simple example

To see example output, run following command in `sample` folder:

``docker run --rm -v `pwd`:/source:ro -v `pwd`/output:/output oiavorskyi/techdoc-md``
